jOperat - Code By Joenix

jOperat是什么？

jOperat是一段用于简化事件绑定流程的，小巧、精致、轻便的javascript代码，你也可以称呼它为“轻框架”，第一版本创始于2013年的《世界程序员日》，同时也是作者的生日。

jOperat不能做什么？

  • 它即不是“插件”，所以不能用它来直接完成业务需求，如焦点图、TAB切换等；
  • 它也不是“UI库”，所以它没有那么多的“视觉表现”，也似乎不那么“紧跟潮流”，HTML5？CSS3？；
  • 它更不是“框架”，所以看上去它并不是那么的全能、彪悍，不能与jQuery、prototype等相提并论。

jOperat擅长做什么？

  1. 开发更便捷：简化事件绑定，并默认支持异步加载的DOM；
  2. 运行更高效：全局冒泡机制，几乎不会因事件操作而产生额外的内存垃圾；
  3. 提升产品稳定性：无须为对象获取异常、重复绑定事件、异步加载DOM等情况做容错处理；
  4. 缩短开发周期：终于可以腾出时间来优化产品体验了！
